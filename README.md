# atelier-vision
Atelier is a sophisticated, multimodal AI assistant designed for high-precision image analysis and professional interaction. Built with a focus on studio aesthetics and socially intelligent conversation, Atelier bridges the gap between visual data and professional insights.
# Key Features
* **Advanced Visual Analysis:** Leverages the Llama 4 Scout model via Groq for near-instant, high-fidelity image interpretation.
* **Studio-Grade UI:** A minimalist, "Atelier-style" interface.
* **Social Intelligence:** Custom-engineered system prompts for professional interactivity, including gratitude acknowledgment and smart exit protocols.
* **Contextual Memory:** Stateful conversation tracking that allows for deep, back-and-forth analysis of visual assets.
* **Refined UX:** Message alignment (User Right / Bot Left) providing a modern, premium chat experience.
# Tech Stack
* **Brain:** Llama 4 Vision (via Groq Cloud)
* **Frontend:** Streamlit (Custom CSS)
* **Language:** Python 3.10+
* **Image Processing:** Pillow (PIL)
